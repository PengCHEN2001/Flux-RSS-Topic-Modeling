Visualisations BERTopic

Résultats de la modélisation thématique // run_bertopic.py

Les visualisations ci-dessous ont été générées avec run_bertopic.py sur le corpus analysé morphosyntaxiquement avec spaCy (corpus_analyse.json, 3 207 documents).

Commandes :

python run_bertopic.py -f json corpus_analyse.json -o bertopic_2d.html --chart 2d
python run_bertopic.py -f json corpus_analyse.json -o bertopic_bar.html --chart barchart
python run_bertopic.py -f json corpus_analyse.json --token lemma --pos NOUN VERB -o bertopic_lemma.html

Options disponibles :

  • --token lemma / --token form — lemmes ou mots-formes
  • --pos NOUN VERB ADJ — filtrage par catégories grammaticales
  • --chart 2d / --chart barchart — type de visualisation
  • -o fichier.html — fichier de sortie
