Visualisations BERTopic + LDA

Résultats de la modélisation thématique // run_bertopic.pyrun_lda.py

Les visualisations ci-dessous ont été générées avec run_bertopic.py et run_lda.py sur un sous-corpus Figaro analysé morphosyntaxiquement avec spaCy (figaro-analyse.xml, ~1103 articles), avec filtrage POS NOUN PROPN ADJ.

Pipeline complète :

python3 rss_parcours.py -c ../donnees/corpus -s figaro --output-file corpus-filtre-figaro.xml --output-format xml python3 analyzers.py corpus-filtre-figaro.xml figaro-analyse.xml -a spacy python3 run_lda.py figaro-analyse.xml --pos NOUN PROPN ADJ --num-topics 8 --passes 20 -c lda_viz_figaro.html python3 run_bertopic.py figaro-analyse.xml -f xml --pos NOUN PROPN ADJ --chart barchart -o bertopic_barchart_figaro.html python3 run_bertopic.py figaro-analyse.xml -f xml --pos NOUN PROPN ADJ --chart 2d -o bertopic_2d_figaro.html python3 run_bertopic.py figaro-analyse.xml -f xml --pos NOUN PROPN ADJ --chart hierarchy -o bertopic_hierarchy_figaro.html python3 run_bertopic.py figaro-analyse.xml -f xml --pos NOUN PROPN ADJ --chart heatmap -o bertopic_heatmap_figaro.html python3 run_bertopic.py figaro-analyse.xml -f xml --pos NOUN PROPN ADJ --chart terms -o bertopic_terms_figaro.html python3 run_bertopic.py figaro-analyse.xml -f xml --pos NOUN PROPN ADJ --chart over_time -o bertopic_over_time_figaro.html
